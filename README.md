# Lillies Wick & Wonder

A luxury candle brand website built with Next.js — featuring a dark, moody aesthetic with gold accents.

---

## Pages

- **Home** — Hero, featured collections, brand story, newsletter signup
- **Shop** — Full product catalog with category filtering
- **Collections** — Collection overview
- **About** — Brand story
- **Contact** — Contact form

## Tech Stack

- [Next.js](https://nextjs.org/) 16 (App Router)
- React 19 + TypeScript
- Tailwind CSS v4
- Framer Motion 12
- Google Fonts — Playfair Display & Inter

## Getting Started

```bash
cd web
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Project Structure

```
Lillies/

│   │   ├── components/   # Reusable UI components
```
